# IoT Dashboard

Real-time IoT sensor monitoring dashboard built with Go, HTMX, and TimescaleDB.

## About
IoT Dashboard provides a live monitoring interface for IoT sensor networks. It displays real-time sensor data via Server-Sent Events (SSE), with time-series storage optimized for high-throughput sensor telemetry.

## Tech Stack
- Backend: Go, Server-Sent Events (SSE)
- Frontend: HTMX, minimal JavaScript
- Database: TimescaleDB (time-series optimized PostgreSQL)
- Visualization: Real-time charts and gauges

## Status
Active development - real-time data pipeline and dashboard UI established.
